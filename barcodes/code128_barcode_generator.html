Code 128 Barcode Generator

Supported characters/codes:

  • Code set A: ASCII codes 0-95 (chars 0-9, A-Z, special, control).
  • Code set B: ASCII codes 32-127 (chars 0-9, A-Z, a-z, special).
  • Code set C: ASCII codes 48-57 (chars 0-9 in double density).

Javascript backslash escapes are supported, e.g. \n, \x0a, \u000a (use \\ to get a literal backslash). FNC codes 1-3 may be specified as \F1, \F2, \F3. Codes for code set switching/shifting and FNC4 are currently not supported, except for auto-switching from code set C to A for the last digit when there's an odd number of digits.
